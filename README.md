# Homicide-1980-2014-Exploratory-Data-Analysis-V-1

## Overview

This project presents an Exploratory Data Analysis (EDA) of a homicide crime dataset using Python. The objective is to uncover meaningful patterns, trends, and relationships within the data through data cleaning, statistical analysis, and visualization.

The analysis focuses on understanding crime distribution across different years, victim and perpetrator demographics, crime resolution rates, and other important factors that may provide insights into homicide cases.

---

## Objectives

- Clean and preprocess the dataset.
- Explore the distribution of numerical and categorical variables.
- Analyze relationships between important features.
- Identify trends in solved and unsolved homicide cases.
- Visualize patterns using informative plots.
- Summarize key findings from the dataset.

---

## Dataset

The dataset contains homicide records with information such as:

- Agencies
- Year
- Victim Age
- Victim Sex
- Perpetrator Age
- Perpetrator Sex
- Crime Solved Status
- Weapon
- Relationship
- State
- Crime Type
- and other related attributes.

## Exploratory Data Analysis

The following analyses were performed:

### Data Cleaning

- Handled missing values
- Removed unnecessary columns
- Corrected data types
- Prepared data for analysis

### Univariate Analysis

- Distribution of victim age
- Distribution of perpetrator age
- Crime solved status
- Victim gender
- Weapon usage
- Relationship distribution
- Crime occurrence by year

### Bivariate Analysis

- Victim Gender vs Crime Solved
- Crime Solved vs Year
- Weapon vs Crime Solved
- Relationship vs Crime Solved
- Victim Age vs Perpetrator Age
- Additional categorical comparisons

### Correlation Analysis

- Correlation Heatmap
- Relationship between numerical variables

---

## Key Findings

- Crime solved rates vary significantly across different years.
- Most numerical variables exhibit weak linear correlations.
- Victim Age and Perpetrator Age show only a weak positive relationship.
- The dataset suggests that categorical variables contribute more information than numerical variables.
